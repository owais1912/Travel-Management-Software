# Travel Management Microservices Platform
A modern, scalable backend solution for real-time cab bookings and passenger profile management.

## 🌟 Strategic Focus
This project demonstrates the transition from monolithic architecture to **Microservices**, focusing on high availability, loose coupling, and automated quality assurance—principles critical for enterprise banking systems.

## 🛠 Tech Stack & Core Concepts
* **Framework:** Spring Boot (Java)
* **Architecture:** Microservices with RESTful API design
* **Testing:** JUnit (Functional and Unit Testing)
* **Build Tool:** Maven
* **Documentation:** API Frameworks

## 🏗 Microservice Components
1. **Passenger Service:** Manages passenger profiles, authentication, and history.
2. **Booking Service:** Orchestrates real-time cab requests and scheduling logic.
3. **API Gateway:** Centralized entry point for various web methods.

## ✅ Quality Assurance
Implemented rigorous **Unit Testing with JUnit** to validate business logic and ensure 100% reliability for passenger booking web methods.
